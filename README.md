# 📋 TrackIt – Stay organized, one task at a time.

TrackIt is a modern, feature-rich, and visually stunning TODO application designed to help you manage your daily tasks with ease. Built with a focus on user experience, it offers a seamless interface, dark mode support, and advanced task management features.

![TrackIt Logo](TrackIT%20copy.jpg)

## ✨ Features

- **📊 Comprehensive Dashboard**: Instantly view your productivity with stats for Total, Completed, Pending, and Overdue tasks.
- **📈 Progress Tracking**: A dynamic progress bar visualizes your overall task completion.
- **🏷️ Smart Categorization**: Organize tasks by category (Work, Personal, Study) and assign priorities (Low, Medium, High).
- **⏰ Smart Deadlines**: Set due dates and get visual alerts for tasks that are "Due Soon" or "Overdue".
- **🔍 Advanced Search & Filter**: Quickly find any task or filter by status to focus on what matters.
- **🌓 Dark Mode**: Switch between a clean light theme and a sleek dark mode for a comfortable experience.
- **👥 User Management**: Supports multiple users with a persistent recent users list.
- **📥 Import/Export**: Easily backup or migrate your tasks using the JSON export and import features.
- **📱 PWA Ready**: Install TrackIt as an app on your device for a native-like experience.
- **🖱️ Drag & Drop**: Intuitively reorder your tasks to prioritize your day.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3 (Tailwind CSS)**: Modern, responsive styling.
- **Vanilla JavaScript**: Core logic and interactivity.
- **LocalStorage**: Persistent data storage without a backend.
- **PWA**: Service workers and manifest for offline and mobile installation.
