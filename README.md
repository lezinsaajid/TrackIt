# 📋 TrackIt – Stay organized, one task at a time.

TrackIt is a modern, feature-rich TODO application designed for visual thinkers and productivity enthusiasts. It offers a seamless, premium interface with real-time stats, smart categorization, and a fully installable PWA experience.

## ✨ Key Features

### 📊 Dynamic Dashboard
Get a high-level view of your productivity. TrackIt features a real-time stats panel that updates instantly:
- **Total Tasks**: Keep track of your complete workload.
- **Completed**: Celebrate your wins with a green success status.
- **Pending**: See exactly what's left on your plate.
- **Overdue**: Catch missed deadlines with bright red alerts.
- **Overall Progress**: A smooth, animated progress bar visualizes your daily completion rate.

### 🏷️ Smart Task Management
- **Categorization**: Organize your life into **Work**, **Personal**, and **Study** categories.
- **Priority Scaling**: Assign **Low**, **Medium**, or **High** priorities with color-coded visual cues.
- **Deadlines**: Set precise due dates with a built-in datetime picker.
- **Drag & Drop**: Intuitively reorder your tasks to plan your day exactly how you want.

### 🌓 Dark Mode & Personalization
- **Seamless Themes**: Switch between a clean Light Mode and a sleek Dark Mode with a single click.
- **Multi-User Support**: Create separate profiles to keep your personal and professional tasks distinct.

### 💾 Data Security & Portability
- **Offline First**: All your data is saved locally using `LocalStorage`—no internet required to manage your day.
- **Import/Export**: Move your data between devices easily with JSON-based export and import functions.

## 🛠️ Technical Deep Dive

TrackIt is built using a modern, lightweight tech stack for maximum performance:
- **Frontend**: Semantic HTML5 and Vanilla JavaScript (ES6+).
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for a responsive, utility-first design system.
- **Persistence**: Browser `LocalStorage` for zero-latency data management.
- **PWA**: Features a `manifest.json` and `Service Worker` for full mobile and desktop installation.

## ⌨️ Keyboard Shortcuts

Speed up your workflow with these built-in shortcuts:
- `Alt + N`: Quickly focus the "Add New Task" field.
- `Alt + S`: Search through your tasks instantly.
- `Alt + T`: Toggle between Light and Dark mode.

## 🏁 Getting Started

1. **Clone the repo**:
   ```bash
   git clone https://github.com/lezinsaajid/TrackIt.git
   ```
2. **Open the App**:
   Simply open `index.html` in any browser or use a local live server.
3. **PWA Installation**:
   Click the **"Install"** icon in your browser's address bar to use TrackIt as a standalone desktop or mobile application.
